SwiftRide Taxi Booking System

A complete taxi booking management system built with Python and Tkinter GUI, developed as a Year 1 university project.

  • User registration and authentication
  • Taxi booking with car selection
  • Real-time booking management
  • Payment simulation system
  • JSON-based data persistence
  • Interactive GUI with Tkinter
Technologies used:
  • Python 3.x -core functionality
  • Tkinter - GUI framework
  • PIL (Pillow) - Image handling
